# emacs - A quick guide to getting started and command shortcuts

## Basics

A quick breakdown for abbreviations:

| Legend         | Notes               |
| -----          | -----               |
| C - Ctrl       | control key         |
| M - Option     | meta key/option key |
| s - super      | command key         |
| S - Shift      | shift key           |
